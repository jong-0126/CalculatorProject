## 프로젝트 소개
자바를 이용하여 계산기를 개발해 보았습니다.

## 개발 기간
2025-02-25 ~

## 개발 환경
- java: 17
- jdk: 17.0.11
- 빌드 도구: Gradle
- IDE: IntelliJ IDEA
- 버전 관리: Git, GitHub

## 주요 기능
- 기본적인 사칙연산(덧셈, 뺄셈, 곱셈, 나눗셈)
- 사용자 인터페이스 방식: 콘솔 기반
- 예외 처리: 0으로 나누기, 숫자를 입력해야하는데 다른 것을 입력, 사칙연산을 입력해야하는데 다른 것 입력
- 첫번째 값 삭제 프로세스 추가
- exit를 누르면 계산기 종료

## Lv1과 Lv2로 나누어 진행
- Lv1: 클래스 없이 기본적인 연산을 수행할 수 있는 계산기 만들기
- Lv2: 클래스를 적용해 기본적인 연산을 수행할 수 있는 계산기 만들기

## 도전 과제: 람다, 스트림, enum 활용
